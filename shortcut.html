Quickly log expenses with a triple tap. Add this shortcut to your iPhone and run it to instantly log a transaction in TapBudget without opening the app.

⚡ Import Shortcut

Opens iCloud Shortcuts. Open the link on your iPhone to add the shortcut.

How to set up the Triple Tap Shortcut
1

Import the shortcut

Tap the "Import Shortcut" button above to add the shortcut to your Shortcuts library.

2

Add your API Key

On your iPhone, after clicking "Set Up Shortcut" you'll see a prompt asking for your API Key. Copy and Paste the API key from your Profile page in the TapBudget app.

3

Navigate to Settings

On your iPhone, navigate to Settings and search "Back Tap". Select "Triple Tap", scroll to the bottom, and select the imported shortcut.
