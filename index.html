Setup Guide
Ready in 60 seconds
1

Import the Shortcut

Tap "Import Shortcut" above. Safari will open the Shortcuts app and add it automatically.

2

Paste your API Key

Open TapBudget → Profile → Integrations & API → tap Copy API Key. Then paste it into the shortcut when prompted.

3

Enable Back Tap

Go to Settings → Accessibility → Touch → Back Tap → Triple Tap and select "TapBudget Triple Tap".

4

Triple tap & done!

Tap the back of your iPhone 3 times, enter the amount, description and category. Transaction logs instantly — no app required.
